******************************例程详细描述*****************************
功能描述:
	本例程演示了如何使用TIM2捕获COMP的输出来测量脉冲宽度。
	将COMP1负端输入配置为DAC1，COMP1正端输入配置为PA1，将COMP1输出映射到TIM2，具体配置如下:
	1、DAC配置为12bit右对齐，DAC输出为output = VDD * DOR / 4095= 3.3 * 2000 / 4095= 1.61V
	2、PA1输入一个外部脉冲信号，其中幅度要大于1.61V，脉冲的持续需要时间满足以下条件：
	不能高于65535 * TIM2CLK = 2.047 ms，不能低于TIM2 ISR processing time / TIM2CLK ~ 110 (clock cycle) / 48 MHz = 2.3 us                                         
 	3、TIM2时钟为48M，TIM2的通道4配置为输入捕获，同时将通道4映射到COMP1，捕获上升和下降沿                                                              
                 最后通过串口打印测量脉冲宽度。                                              
时钟配置:
	本例程采用HSE+PLL48MHz作为系统时钟，用户如果需要更改系统时钟配置，请参考对应用户手册或者
	RCC库函数及RCC相关例程。

适用芯片:
	HK32L0xx全系列

涉及外设:
	COMP/TIM2/GPIO/DAC/USART
	
涉及管脚:	
	COMP1 正端输入 -------------> PA1
	COMP1 负端输入 -------------> PA4（DAC1）
	UART3 RX -----------> PA5
	UART3 TX -----------> PA4
注意事项:
	1. 推荐使用MDK5.00及其以上版本编译本工程（MDK-ARM）
	          或者IAR8.22及其以上版本编译本工程(EWARM);
	2. 使用之前检测电源VCC、GND等是否正常。
如何使用:
	1. 直接编译程序下载到评估版，重启评估版;
	2. 在PA1上输入满足条件的脉冲波形，观察串口打印情况。
