******************************例程详细描述*****************************
功能描述:
        本例程描述了CLU如何使用，包括CLU的初始化，映射到GPIO的程序编写，帮助
        用户学会使用CLU的D触发器功能。

        在本例程对输入的波形进行二分频，借助UART3去实现人机对接的界面，提示用
        户的每一步操作，用户可以根据自己需求灵活更改。
	
时钟配置:
        本例程采用HSE48MHz作为系统时钟，用户如果需要更改系统时钟配置，可通过
        SYSCLK_SOURCE宏定义来选择不同的时钟源。

适用芯片:
        HK32L0xx全系列

涉及外设:
        UART3 / GPIO / CLU
	
涉及管脚:
	UART3 Pin：
        UART3_TX  ------------>  PA4
        UART3_RX  ------------>  PA5
	CLUx Pin:
        CLU0_MXA  ------------>  PA1
        CLU0_OUT  ------------>  PF4
        CLU3_MXA  ------------>  PA12

    GPIO Pin:
		Output: PB0     	

注意事项:
        1. 推荐使用MDK5.00及其以上版本编译本工程（MDK-ARM）
           或者IAR8.22及其以上版本编译本工程(EWARM);
        2. 使用之前检测电源VCC、GND等是否正常。

如何使用:
        1. 直接编译程序下载到评估版，重启评估版;
        2. 通过串口打印确认CLU是否正常初始化;
        3. 根据输出的内容，硬件跳线连接GPIO pin脚，并通过串口回应要求数字；
        4. 查看最终输出结果，若硬线跳线正确则通过串口打印正确结果，否则打印错误结果；
        5. 建议单独调试并同时查看寄存器值来理解CLU的功能。
