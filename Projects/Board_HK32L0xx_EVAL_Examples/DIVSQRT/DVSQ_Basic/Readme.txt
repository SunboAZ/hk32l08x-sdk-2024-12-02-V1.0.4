******************************例程详细描述*****************************
功能描述:
	本例演示如使用除法开方运算单元（DVSQ）。

	本例程演示了无符号除法运算，开方运算（结果为整数）和高精度开
	方运算（结果含小数部分）。

时钟配置:
	本例程采用HSI48MHz作为系统时钟，用户如果需要更改系统时钟配置，请参考对应用户手册或者
	RCC库函数及RCC相关例程。

适用芯片:
	HK32L0xx全系列

涉及外设:
	RCC
	UART3
	GPIO
	DVSQ

涉及管脚:
	PA4
	PA5

注意事项:
	1. 推荐使用MDK5.00及其以上版本编译本工程（MDK - ARM）
	或者IAR8.22及其以上版本编译本工程(EWARM);
	2. 使用之前检测电源VCC、GND等是否正常。
如何使用:
	1. 直接编译程序下载到评估版，重启评估版;
	2. 通过LED1亮灯，串口打印确认DVSQ是否正常工作;
