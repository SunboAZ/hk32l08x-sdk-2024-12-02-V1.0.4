******************************例程详细描述*****************************
功能描述:
    本例程描述了DMA的外设到内存以及内存到外设传输，传输的开始是由软件触发。
    1.首先通过内存到外设将TxBuffer1搬运到USART1_TX
    2.然后通过外设到内存从USART2_RX搬运数据到RxBuffer2
    3.接着通过内存到外设将TxBuffer2搬运到USART2_TX
    4.最后通过外设到内存将USART1_RX搬运数据到RxBuffer1
    5.搬运完成以后比较RxBuffer1和TxBuffer2的数据以及RxBuffer2和TxBuffer1的数据，如果都相同，则表示搬运成功。

时钟配置:
    本例程采用HSE + PLL48MHz作为系统时钟，用户如果需要更改系统时钟配置，请参考对应用户手册或者
    RCC库函数及RCC相关例程。

适用芯片:
    HK32L0xx全系列

涉及外设:
    DMA / RCC / GPIO /USART

涉及管脚:
    LED1(D2) : PF4
    USART1_TX: PA9
    USART1_RX: PA10
    USART2_TX: PA2
    USART2_RX: PA3

注意事项:
    1. 推荐使用MDK5.00及其以上版本编译本工程（MDK - ARM）
    或者IAR8.22及其以上版本编译本工程(EWARM);
    2. 使用之前检测电源VCC、GND等是否正常;
    3. 本例程中需要将PA9与PA3相连，PA10与PA2相连，
    4. 本例程用到的PA2、PA3、PA9、PA10均为LCD的功能引脚，所以用本例程时不能接入LCD屏。
如何使用:
    1. 直接编译程序下载到评估版，重启评估版;
    2. 程序上电会自动进行测试，测试通过则D2会闪烁，表明DMA从内存到外设以及从外设到内存工作正常，
    如果D2常量则表示DMA搬运异常。
