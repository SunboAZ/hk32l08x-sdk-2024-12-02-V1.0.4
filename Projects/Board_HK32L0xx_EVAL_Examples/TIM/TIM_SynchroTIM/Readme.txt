******************************例程详细描述*****************************
功能描述:
	本例程展示如何在并行模式下同步TIM1和定时器（TIM3和TIM2）。
	1. 配置TIM1作为主计时器，PWM输出功能被使能，TIM1计数器更新作为触发输出；
	2. 配置TIM3作为TIM1的从模式，PWM功能被使能，TIM1的更新事件 通过ITR0触发TIM3的计数器，TIM3配置为门控模式；
	3. 配置TIM2作为TIM1的从模式，PWM功能被使能，TIM1的更新事件 通过ITR0触发TIM2的计数器，TIM2配置为门控模式。
	4. TIM1配置了重复计数器。

时钟配置:
	本例程采用HSE48MHz作为系统时钟，用户如果需要更改系统时钟配置, 请参考对应用户手册或者
	RCC库函数及RCC相关例程。

适用芯片:
	HK32L0XX全系列

涉及外设:
	TIM,GPIO
	
涉及管脚:	
	TIM1_CH1(PA8),    TIM3_CH1(PC6),     TIM2_CH1(PA0)

注意事项:
	1. 推荐使用MDK5.00及其以上版本编译本工程（MDK-ARM）
	    或者IAR8.22及其以上版本编译本工程(EWARM);
	2. 使用之前检测电源VCC、GND等是否正常。

如何使用:
	1. 直接编译程序下载到评估版，重启评估版;
	2. 用示波器查看TIM1_CH1(PA8), TIM3_CH1(PC6), TIM2_CH1(PA0)的PWM输出，占空比依次为50%，33.3%，50%。
