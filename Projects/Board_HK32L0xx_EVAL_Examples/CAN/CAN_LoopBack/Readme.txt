******************************例程详细描述*****************************
功能描述:
	本例程描述了CAN在LoopBack模式下的发送和接收。
	通过调用库函数CAN_Init()和CAN_FilterInit()来完成对CAN的配置，先发送一帧波特率为125kbps的标准帧，若LED1灭，则接收失败；LED1亮，则接收成功。
	接着发送一帧波特率为500kbps的扩展帧，若LED2灭，则接收失败；LED2亮，则接收成功。

时钟配置:
	本例程采用HSE+PLL48MHz作为系统时钟，用户如果需要更改系统时钟配置，请参考对应用户手册或者
	RCC库函数及RCC相关例程。

适用芯片:
	HK32L0xx全系列

涉及外设:
	CAN/GPIO
	
涉及管脚:	
	LED1 -----------> PF4
	LED2 -----------> PF5
	CAN RX -----------> PA11
	CAN TX -----------> PA12

注意事项:
	1. 推荐使用MDK5.00及其以上版本编译本工程（MDK-ARM）
	          或者IAR8.22及其以上版本编译本工程(EWARM);
	2. 使用之前检测电源VCC、GND等是否正常。
如何使用:
	1. 直接编译程序下载到评估版，重启评估版;
	2. 观察LED1来判断是否成功接收标准帧，观察LED2来判断是否成功接收扩展帧
