******************************例程详细描述*****************************
功能描述:
	本例程演示了如何使用RTC ALARM自动唤醒STOP和EXTI唤醒STOP。
	程序运行后，RTC被配置为生成ALARM事件，然后系统进入STOP模式。要从STOP模式唤醒，可以按下KEY2，产生EXTI中断唤醒，
	否则RTC ALARM将在5秒内唤醒系统。退出STOP后，系统时钟被重新配置为其先前状态（因为HSE和PLL在STOP模式下被禁用）。
	配置EXTI line 17(RTC ALARM)中断，在中断函数中点亮LED1。配置KEY2为EXTI模式，在EXTI中断函数中点亮LED2。                        	
                                              
时钟配置:
	本例程采用HSE+PLL48MHz作为系统时钟，用户如果需要更改系统时钟配置，请参考对应用户手册或者
	RCC库函数及RCC相关例程。

适用芯片:
	HK32L0xx全系列

涉及外设:
	PWR/EXTI/GPIO
	
涉及管脚:	
	KEY2 -----------> PC13
	LED1 -----------> PF4
	LED2 -----------> PF5
	UART3 RX -----------> PA5
	UART3 TX -----------> PA4
注意事项:
	1. 推荐使用MDK5.00及其以上版本编译本工程（MDK-ARM）
	          或者IAR8.22及其以上版本编译本工程(EWARM);
	2. 使用之前检测电源VCC、GND等是否正常。
如何使用:
	1. 直接编译程序下载到评估版，重启评估版;
	2. 调节VDD电压，观察LED1亮灭情况。
